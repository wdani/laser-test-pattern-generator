Laser Test Pattern Generator v1.6.1
===================================

This is a small backend/API groundwork and test-hardening release for the
stable Python/Tkinter v1.x line.

The generated MKS and generic NC toolpath behavior is unchanged.

Highlights
----------

- Cross-platform Python/Tkinter GUI and command-line generator.
- Makera Studio .mks generation.
- Generic .nc / G-code generation.
- NC power profiles: Makera (0-1), GRBL (0-1000), 8-bit (0-255), and Custom.
- Default generic NC profile is Makera (0-1).
- JSON CLI API groundwork for future UI/frontend work:
  - python makera_material_test_generator.py --api app-info
  - python makera_material_test_generator.py --api default-settings
- Python core settings and the JSON CLI API remain the source of truth for
  future UI/frontend work.
- app-info reports release metadata, supported output formats, and available
  API commands.
- default-settings reports the full default GeneratorSettings values as valid
  JSON.
- Added and expanded tests for app-info, default-settings, and generated output
  validation.
- No major Tkinter UI rewrite is included in this release.
- Future larger UI work is expected to target Tauri v2.0, while v1.x remains
  the stable Python/Tkinter version.

Existing v1.x Features
----------------------

- Modern Tkinter/ttk Light and Dark themes.
- Improved numeric controls for common grid, stock, speed, power, laser, and
  label settings.
- Right-side quick preview on wide windows.
- Automatic preview refresh after generation, preset loads, tab changes, and
  key layout edits.
- Preset manager with metadata fields, import/export, and clearer save
  behavior.
- Optional preset metadata: name, material, machine, laser module, notes,
  safety note, and reference image.
- Conservative preset starting points for cork, wood, cardboard, and
  Makera-style generic NC tests.
- Output modes for MKS, NC, or both.
- English GUI.
- Generated labels selectable between English and German.
- Included Makera templates and JSON presets.
- Windows convenience launcher: start_gui_windows.bat.

Linux/macOS Start
-----------------

python3 run_gui.py

or:

python3 makera_material_test_generator.py --gui

Windows Start
-------------

Double-click start_gui_windows.bat, or run:

python makera_material_test_generator.py --gui

Notes
-----

- Windows executable builds remain experimental.
- No third-party Python packages are required.
- The right-side preview is approximate and does not replace verification in
  Makera Studio or an NC sender/viewer.
- Makera profile maps 20% power to S0.2 and 40% power to S0.4.
- GRBL profile maps 20% power to S200 and 40% power to S400.
- Presets are starting points, not guaranteed safe machine settings.
- Always preview generated .mks files in Makera Studio after Recalculate.
- Always verify generic .nc files against your controller's S-value scale.
