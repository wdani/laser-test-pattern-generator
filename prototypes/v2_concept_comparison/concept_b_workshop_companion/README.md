# Concept B - Workshop Companion Command Center

Open `index.html` directly.

This concept treats the v2 UI as a modular CNC/Laser Workshop Companion. Active workflows are presented as current workshop jobs, while calculators, checklists, journals, and diagnostics are supporting helpers for the selected job.

The two active workflow jobs are:

- Laser Test Pattern Generator
- Parametric Design / Nameplate Generator

This is still static only: no backend calls, no generation, no dependencies, and no machine-control output.
