# Changelog

## v1.0

First usable release package.

### Added

- Tabbed GUI
- Makera Studio MKS output
- generic NC output
- MKS, NC, and Both output modes
- preset manager
- included presets
- auto filename generation
- overwrite protection
- auto positioning inside stock
- approximate preview tab
- English GUI
- English/German generated label language option
- Windows launcher batch file

### Notes

This version should be treated as a first stable prototype. Generated files must still be previewed before use.
